## ğŸ® How to Use

### Button Controls
- **Start**: Begin timing
- **Pause**: Pause the current time
- **Reset**: Reset timer to 00:00:00:000 and clear lap times
- **Lap**: Record current time as a lap (only available when running)

### Keyboard Shortcuts
- **Space**: Start/Pause the stopwatch
- **R**: Reset the stopwatch
- **L**: Record a lap time
